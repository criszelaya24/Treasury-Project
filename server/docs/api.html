  • Request
  • Response

Headers

  • Authorization: required(string)

    Auth token with the credential returned

Body

Media type: application/json

Type: object

Properties
  • numero: required(int)

    numero from factura

  • proveedor: required(int)
  • currency: required(int)

    ID from currency

  • monto: required(int)

    Monto from factura

  • detalle: (string)

    details from factura

  • fecha: required(date)

    created date from factura / format yyyy-mm-dd hh:mm:ss

  • vencimiento: required(date)

    expiration date from factura / format yyyy-mm-dd hh:mm:ss

  • status: required(int)

    id status from factura

  • empresa: required(int)

    id from empresa link to factura

{
      "numero": 2019,

      "currency": 1,
      "monto": 10000,
      "detalle": "Pago internet",
      "fecha": "2019-12-11 23:31:30",

      "status": 1,
      "empresa": 3
}
